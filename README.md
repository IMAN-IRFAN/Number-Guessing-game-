# Number Guessing Game 🎮

A multi-level number guessing game with progressive difficulty.

## Features
- 10 levels per round (increasing difficulty)
- Scoring system (+10 for correct, -5 penalty)
- Intelligent hints (distance-based)
- Maze progress visualization
- Time limit on advanced levels (10+)
- Score tracking across rounds

## How to Run
```bash
javac number_guessing.java
java number_guessing
```

## Game Mechanics
- Guess secret numbers in given range
- Limited chances per level
- Hints guide you closer
- Level 10+ has 30-second time limit
